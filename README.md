# Image Trim (Chrome Extension)

画像をアップロードすると、四辺の余白（白 or 透明）だけを自動でカットしてダウンロードできます。

## 使い方

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択
4. 拡張機能アイコンをクリックして画像をアップロード
5. 必要なら「許容誤差」を調整してダウンロード

## 仕組み

- 画像の外周（上/下/左/右）をサンプリングして背景色を推定
- 透明な余白が多い場合は透明背景として扱う
- 背景色と十分に離れている画素だけを「本体」とみなし、その範囲でトリミング

## ファイル構成

- `manifest.json` Chrome拡張の定義
- `popup.html` UI
- `popup.css` デザイン
- `popup.js` 画像解析とトリミング処理
